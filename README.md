# Gönül Köprüsü 🌉

İyiliğe açılan köprü — yardıma ihtiyacı olanlarla iyilik yapmak isteyenleri buluşturan gönüllü dayanışma platformunun web sitesi.

## Yapı

```
.
├── index.html      # Ana sayfa
├── css/style.css   # Tüm stiller (responsive + animasyonlar)
└── js/main.js      # Etkileşimler (menü, sayaçlar, formlar)
```

## Yerelde Çalıştırma

Herhangi bir derleme adımı yoktur; statik bir sitedir.

```bash
python3 -m http.server 8000
# Tarayıcıda http://localhost:8000 adresini açın
```

## Özellikler

- Modern, sıcak renk paletiyle responsive tasarım (mobil + masaüstü)
- Hero bölümü, animasyonlu istatistik sayaçları ve yüzen kampanya kartları
- Hakkımızda, Nasıl Çalışır, Aktif Kampanyalar, Gönüllü Ol, Tanıklıklar ve İletişim bölümleri
- Scroll-reveal animasyonları ve mobil hamburger menü
- Gönüllü başvuru ve iletişim formları (demo doğrulama ile)
